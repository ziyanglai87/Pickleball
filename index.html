Setup Steps

  1. Create a Google Sheet with 3 tabs: Players, Matches, Settings.
  2. Open Extensions → Apps Script.
  3. Paste the Apps Script code from the file provided with this dashboard.
  4. Deploy as Web app. Execute as yourself. Access: anyone with the link.
  5. Paste the deployment /exec URL here and click Save URL.

Sync rule: last save wins. Use Load from Cloud before making edits if someone else may have updated the sheet.

Mock Rating Model — Provisional Point-Share Version

Expected point share = 1 / (1 + 10^((Opponent Team Rating - Your Team Rating) / Scale)). Actual point share = team points / total points. Early/unrated players use the higher Provisional K, then taper toward Mature K.

Calibration reference: Two unrated 3.500-seeded players losing 11–15 to a 3.343/3.013 pair, then 5–15 to a 3.460/3.643 pair should land around 3.10–3.12 with default settings.
